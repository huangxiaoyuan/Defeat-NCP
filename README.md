在重点人员隔离过程中，社区人员以及医护人员在固定时间需要到隔离地点对隔离人员或者患者进行体温测量，（在武汉已经开始不落一户、不漏一人进行体温排查），这大大增大了工作人员的感染的风险，工作量也及其巨大，消耗巨大的人力物力和财力。
同时当前各地陆续开始进行复工，企事业单位都需要对进入人员进行体温测量和统计，因上班人员多，测量和统计任务极其繁重。
